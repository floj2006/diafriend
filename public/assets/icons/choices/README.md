# Choice Icons

Сюда складываем иконки именно для вариантов выбора внутри сцен.

Папки:
- `intro/` — большие кнопки `да` / `нет`
- `breakfast/` — еда для завтрака
- `school/` — вещи для рюкзачка
- `playground/` — сок, вода, печенье, бег
- `evening/` — шаги вечерней сцены

Примеры файлов:
- `intro/yes.svg`
- `breakfast/porridge.svg`
- `breakfast/water.svg`
- `school/meter.svg`
- `playground/juice.svg`
- `evening/measure.svg`

Сейчас в папках уже лежат:
- простые `svg` для кнопок `да / нет`
- нарезанные `png` из общего листа иконок для еды, школы и прогулки

Если захочешь, их можно заменить на новые нарисованные файлы с теми же именами.
