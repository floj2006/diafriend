# Assets

Здесь лежат все будущие отрисованные и озвученные материалы проекта.

## Иконки

Папка: `public/assets/icons`

- `mascots/` — персонажи: `glukosha.png`, `lada.png`, `tima.png`, `iriska.png`
- `items/` — предметы и сцены: `porridge.png`, `juice.png`, `backpack.png`, `meter.png`
- `ui/` — кнопки, наклейки, бабблы, маленькие декоративные элементы
- `choices/` — иконки именно для вариантов выбора внутри сцен

## Озвучка

Папка: `public/assets/voice/ru`

- `mascots/` — голосовые реплики персонажей
- `scenes/` — озвучка сцен и маленьких побед
- `ui/` — клики, наклейки, мягкие звуки интерфейса
